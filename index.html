    Limechat for OS X Solarized theme

    Project maintained by tadityar Hosted on GitHub Pages — Theme by mattgraham

    What this is.

    This is a theme for an Open Source IRC client for Mac OS X, Limechat. This theme is based on the solarized swatch by Ethan Schoonover and some off-swatch colors. I actually got inspired to create this theme because of the prolonged use of the solarized theme in Sublime Text 3 ;)

    How To Use

    • Install Limechat Like of course huh? You can download it here if you haven't though.
    • Place the theme in the approriate folder Download the theme from links above (either the zip or tar file) and copy the extracted files. Open the Applications folder and find Limechat there, right-click on it, and choose Show Packages Content. Go to Contents > Themes and paste the copied files there.
    • Enable the theme If your Limechat app is already running, restart the app and click Limechat> Preferences > Theme and change the Current Theme to Solarized. That's it!
